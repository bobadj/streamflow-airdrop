# Streamflow Airdrops

#### An Streamflow airdrops explorer built with **Vite**, using **@solana/web3.js**, **Streamflow SDK**. Frontend data fetching and caching are powered by **React Query**.

## Getting Started

### 1. Install Dependencies

```bash
pnpm install
```

### 2. Start Development Server

```bash
pnpm run dev
```

## Project structure

```bash
/src
├── components/         # Shared and reusable UI components
├── contexts/           # app contexts
├── hooks/              # react hooks
├── lib/
├── pages/
├── routes/
├── services/
│   └── streamflow.service.ts
└── utils/            # utils

```

## Tech Stack

- Vite react
- @solana/web3.js
- Streamflow SDK
- TypeScript
- RTK Query
- Tailwind CSS
